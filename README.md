# SELMARK-PMP

El proyecto PMP (Plan Maestro de Planificación) tiene como objetivo la automatización y optimización del proceso de previsión de la producción de prendas en base a la demanda específica histórica de cada tipo de prenda, la popularidad de la colección, la estacionalidad y la perspectivas de crecimiento de
la propia marca, entre otros parámetros.

Esto se aborda mediante la aplicación de algoritmos de Inteligencia Artificial y análisis masivo de datos para la generación de previsiones de venta más precisas.

El reto se aborda desde tres niveles de analítica:

1. Analítica descriptiva, conocer las ventas y productos más demandandos históricamente.
2. Analítica diagnóstica, explicar las ventas en función de parámetros como la colección, precios, ofertas y el calendario de cada año.
3. Analítica predictiva, predecir ventas de las colecciones del próximo año en base a modelos desarrollados con datos históricos.

Este repositorio incluye aquellas utilidades software desarrolladas en el marco del proyecto y liberadas como software libre para el servicio a la comunidad.
